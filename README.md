> *Solving India's real government hospital problem — no more ghanton ki line!*

A full-stack, real-time hospital token management system. Patients book OPD tokens online, track live queue position, and get WhatsApp/SMS alerts when their turn approaches.

|---------|-----------|

| 🔐 Role-based Auth (Patient/Doctor/Admin) | Clerk |

| 📱 SMS + WhatsApp Notifications | Twilio |
